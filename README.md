## 📖 项目介绍 (About The Game)

**My University** 是一款受《双点校园》、《模拟城市》等经典游戏启发的模拟经营游戏。玩家将扮演一名新上任的大学校长，从一片荒地开始，规划校园布局，招聘顶尖教授，管理学校财务，并处理各种突发的校园事件。

我们的目标是创建一个高度可定制、模组友好（Mod-friendly）且具有深度的模拟经营系统。

### 🎮 核心玩法

*   **🏗️ 自由建造**：自由规划教学楼、宿舍、实验室、图书馆和食堂。
*   **👨‍🏫 人员管理**：招聘教授、清洁工和行政人员。关注他们的心情和薪资要求。
*   **🎓 学生培养**：排课、设立奖学金、处理学生投诉。保证毕业率是首要任务！
*   **💰 财务经营**：通过学费、政府拨款和校友捐赠来维持收支平衡。
*   **🌲 随机事件**：应对流感爆发、校园恋爱、学术丑闻或突如其来的教育部检查。

## 🚀 快速开始 (Getting Started)

### 对于玩家

1. 前往 [Releases](https://github.com/YourUsername/My-University/releases) 页面。
2. 下载对应系统的最新版本压缩包。
3. 解压并运行 `MyUniversity.exe`。

### 对于开发者

如果你想参与开发或自行编译项目，请按照以下步骤操作：
